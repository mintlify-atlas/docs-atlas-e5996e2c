## What is Mango Damage Detection?

Mango Damage Detection is a web application that uses a pre-trained TensorFlow model to identify and classify damage on mangoes. The system detects five different types of conditions including diseases, pests, and healthy fruit.

Built with Django for the web interface and FastAPI for model serving, this application provides both a user-friendly web interface and a REST API for automated damage detection.

## How It Works

The application uses a two-tier architecture:

1. **Django Frontend**: Provides a web interface where users can upload mango images
2. **FastAPI Backend**: Serves the TensorFlow model and performs image classification

When you upload an image, the Django application forwards it to the FastAPI service, which preprocesses the image (resize to 224x224, normalize pixel values) and runs inference using the trained model. The prediction is returned to the user with the detected damage type.
